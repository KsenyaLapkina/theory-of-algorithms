# Генератор мемов

## Описание
Приложение для создание мемов. Можно загрузить свое изображение и написать любой текст

## Технологии
- **Python 3.13** 
- **ООП** 
- **Docker** - контейнеризация приложения
- **pytest** - модульное тестирование


## Основные классы
- Database - Класс для работы с SQLite базой данных, хранит информацию о мемах и статистику действий.
- MemeCreator/MemeGenerator - Класс для создания мемов с текстом поверх изображений с использованием библиотеки Pillow.
- FastAPI App - Основной класс веб-приложения FastAPI, обрабатывает HTTP запросы и маршрутизацию.
- HTMLResponse - Класс FastAPI для возврата HTML контента на главную страницу приложения.
- JSONResponse - Класс FastAPI для возврата данных в формате JSON (статистика, результаты создания мемов).
- FileResponse - Класс FastAPI для отдачи файлов (скачивание созданных мемов).
- UploadFile - Класс FastAPI для обработки загружаемых файлов изображений.
- HTTPException - Класс для обработки и возврата ошибок HTTP (валидация файлов, ошибки БД).
- UPLOAD_DIR - Константа/класс пути к директории для хранения загруженных файлов.
- DB_PATH - Константа/класс пути к файлу базы данных SQLite.
- MemeEntity - Сущность/класс данных для хранения информации о конкретном меме (ID, путь к файлу, текст).
- StatsEntity - Сущность/класс данных для хранения статистических записей о действиях пользователей.

## Особенности
- Модульная архитектура: Пакет clothing с тремя модулями
- ООП подход: Наследование, абстрактные классы, инкапсуляция
- Экспорт результатов: Сохранение в форматы Word и Excel
- История расчетов: Автоматическое сохранение в базу данных SQLite
- Docker-контейнеризация: Готовая сборка приложения

## Обычный запуск 

### Создание виртуального окружения
python -m venv venv
venv\Scripts\activate     # Windows

### Установка зависимостей
pip install -r requirements.txt

### Запуск тестов
pytest tests/tests.py -v   

### Запуск приложения
python main.py

### Запуск web-версии
python main.py
### Сборка и запуск в Docker
docker build -f Dockerfile.dev -t meme-generator-dev .
docker run -p 3471:3471 -v .:/app meme-generator-dev
