# Отчет по лабораторной работе 1

**Технологии**  
- Python 3.13
- PyTest 8.4.2

## Задание
Выполнить 11 заданий, инкапсулировать логику в классы, написать тесты, создать отчет.

## Описание проделанной работы
1. Создана модульная структура проекта
2. Реализованы 11 заданий в отдельных модулях
3. Каждое задание инкапсулировано в класс
4. Создан главный управляющий модуль
5. Написаны базовые тесты

## Запуск
```bash
# Клонирование репозитория
git clone <repository-url>
cd lab1

# Создание виртуального окружения
python -m venv venv

# Активация окружения (Windows)
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py

## Запуск тестов
```bash
# Запуск всех тестов
python -m pytest tests/ -v
