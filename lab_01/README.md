## Задание
1)	Установите python. Желательно версии 3.8.10, 3.9.
2)	Создайте репозиторий для дисциплины на GitHub.
3)	Склонируйте его себе на ПК.
4)	Скачайте архив и распакуйте его в свой репозиторий. В нём 11 заданий, которые вам нужно выполнить.
5)	Напишите отчёт c титульным листом по структуре ниже. В отчете код и скриншоты выполнения.
6)	В программе должно быть ReadMe.md и requirements.txt.
7)	Напишите верхнеуровневый модуль, который будет использовать логику из модулей-заданий. Перед этим нужно будет придумать способ инкапсулировать логику для корректного импортирования.
8)	Покройте реализованную функциональность тестами с использованием pytest.

## Описание заданий
1. Дано значение радиуса круга. Выведите на консоль значение площади этого круга с точностью до 4-х знаков после запятой. Далее, пусть есть координаты точки. Если точка point лежит внутри того самого круга [центр в начале координат (0, 0), radius = 42], то выведите на консоль True, Или False, если точка лежит вовне круга. Если точка point_2 лежит внутри круга (radius = 42), то выведите на консоль True, или False, если точка лежит вовне круга.
2. Расставьте знаки операций "плюс", "минус", "умножение" и скобки между числами "1 2 3 4 5" так, чтобы получилось число "25".
3. Есть строка с перечислением фильмов. Выведите на консоль с помощью индексации строки, последовательно:
1)	первый фильм
2)	последний
3)	второй
4)	второй с конца
Запятая не должна выводиться.  Переопределять my_favorite_movies нельзя. Использовать .split() или .find() или другие методы строки нельзя.
4. Создайте списки “моя семья” и список приблизительного роста членов вашей семьи. Выведете на консоль рост отца, выведете на консоль общий рост вашей семьи как сумму ростов всех членов.
5. Есть список животных в зоопарке. Выполните по очереди:
1)	посадите медведя (bear) между львом и кенгуру
2)	добавьте птиц из списка birds в последние клетки зоопарка
3)	уберите слона
4)	выведите на консоль в какой клетке сидит лев (lion) и жаворонок (lark)
6. Есть список песен группы Depeche Mode со временем звучания с точностью до долей минут. Распечатайте общее время звучания трех песен: 'Halo', 'Enjoy the Silence' и 'Clean' в формате: «три песни звучат ХХХ.XX минут».
Есть словарь песен группы Depeche Mode. Распечатайте общее время звучания трех песен: 'Sweetest Perfection', 'Policy of Truth' и 'Blue Dress' в формате «А другие три песни звучат ХХХ минут»
7. Есть зашифрованное сообщение. Нужно его расшифровать и вывести на консоль в удобочитаемом виде.
8. Создайте множество цветов, произрастающих в саду и на лугу. Выведите на консоль:
1)	все виды цветов
2)	те цветы, которые растут и там, и там
3)	те цветы, которые растут в саду, но не растут на лугу
4)	те цветы, которые растут на лугу, но не растут в саду
9. Есть словарь магазинов с распродажами. Создайте словарь цен на продукты следующего вида. Указать надо только по 2 магазина с минимальными ценами
10.  Есть словарь кодов товаров. Есть словарь списков количества товаров на складе. Рассчитать на какую сумму лежит каждого товара на складе. Вывести стоимость каждого вида товара на складе:
1)	один раз распечатать сколько всего столов и их общая стоимость,
2)	один раз распечатать сколько всего стульев и их общая стоимость, и т.д. на складе
