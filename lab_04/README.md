# Лабораторная работа №4

## Описание
Пеб-приложение на Python с использованием FastAPI для выполнения трех алгоритмических задач:
Генерация всех сочетаний из трех цифр (0-9)
Вычисление значений линейной функции f(x) = -1.5x + 2
Сортировка словаря по убыванию ключей

## Технологии
- **Python 3.13** 
- **ООП** 
- **Docker** - контейнеризация приложения
- **pytest** - модульное тестирование


## Основные классы
- CombinationGenerator - генерация трехзначных комбинаций цифр
- FunctionCalculator - вычисление значений математической функции
- DictionarySorter - сортировка словаря по убыванию ключей
- TaskExecutor - координация выполнения всех заданий
- ResultFormatter - форматирование результатов для вывода

## Особенности
- Модульная архитектура: Пакет clothing с тремя модулями
- ООП подход: Наследование, абстрактные классы, инкапсуляция
- Экспорт результатов: Сохранение в форматы Word и Excel
- История расчетов: Автоматическое сохранение в базу данных SQLite
- Docker-контейнеризация: Готовая сборка приложения

 

### Создание виртуального окружения
python -m venv venv
venv\Scripts\activate     # Windows

### Установка зависимостей
pip install fastapi uvicorn jinja2

### Запуск тестов
python -m pytest tests/ -v  

### Запуск приложения
python run_web.py


### Сборка и запуск в Docker
docker build -t algorithms-app .
docker run -p 5629:5629 algorithms-app
