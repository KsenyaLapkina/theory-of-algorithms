# Калькулятор пошива одежды

## Описание
Профессиональное приложение для расчета расхода ткани и стоимости пошива одежды. Программа позволяет рассчитать параметры для пиджаков, брюк различных типов и костюмов-троек.

## Технологии
- **Python 3.13** 
- **ООП** 
- **Docker** - контейнеризация приложения
- **pytest** - модульное тестирование
- **python-docx** - работа с документами Word
- **openpyxl** - работа с Excel файлами

## Основные классы
- arment - Абстрактный базовый класс для изделий
- Jacket - Класс для расчета параметров пиджака
- Pants - Класс для расчета параметров брюк различных типов
- ThreePieceSuit - Класс для расчета костюма-тройки
- ClothingDatabase - Менеджер базы данных SQLite

## Особенности
- Модульная архитектура: Пакет clothing с тремя модулями
- ООП подход: Наследование, абстрактные классы, инкапсуляция
- Экспорт результатов: Сохранение в форматы Word и Excel
- История расчетов: Автоматическое сохранение в базу данных SQLite
- Docker-контейнеризация: Готовая сборка приложения

## Обычный запуск 
# Клонирование и установка
git clone <repository-url>
cd lab_02

# Создание виртуального окружения
python -m venv venv
venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестов
python -m pytest tests/ -v

# Запуск приложения
python main.py

# Сборка и запуск в Docker
docker build -t clothing-calculator .
docker run -it --rm clothing-calculator
